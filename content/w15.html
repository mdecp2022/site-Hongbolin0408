1.為何學習計算機程式需要建立個人倉儲?

A:紀錄每次的更新改版

2.全球資訊網能夠提供甚麼功能或有什麼優點?

A:更方便的網路服務的系統

3.Replit, stud.cycu.org 與 localhost 分別代表甚麼?

A:Replit是提供線上集成開發環境服務的公司，由Amjad Masad與Hata Odeh於2016年共同創立stud.cycu.org注意事項 更新前儘量利用 git pull 與 github 倉儲保持同步localhost是一個在電腦網路中用於表示「此電腦」的主機名。它被用於通過本地環回網路介面，來存取本機執行的服務，並且將會繞過任何物理網路介面硬體。

4.https, ssh 到底提供使用者那些功能或優點?

A:安全外殼協定（Secure Shell Protocol，簡稱SSH）是一種加密的網路傳輸協定，可在不安全的網路中為網路服務提供安全的傳輸環境。

5.Brython 與 Python 有甚麼差別?

A:Python為一種程式語言，Brython能在瀏覽器上用Python取代Java

6.能夠直接在瀏覽器中以 Brython 繪製中華民國國旗, 或模擬綠色方塊在方塊格點中任意移動, 代表甚麼?

A:可以改變圖形的顏色和形狀。
